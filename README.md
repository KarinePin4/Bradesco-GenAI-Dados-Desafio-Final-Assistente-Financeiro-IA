# Bradesco-GenAI-Dados-Desafio-Final-Assistente-Financeiro-IA

Assistente virtual especializado em controle financeiro pessoal. Um dos problemas mais comuns é a falta de hábito de organizar as finanças: a maioria dos brasileiros não registra seus gastos diários e, ao final do mês, não consegue prever para onde foi o dinheiro.
Este assistente foi treinado com metodologias de agilidade corporativa, utilizando principalmente práticas de acompanhamento contínuo, como perguntas frequentes inspiradas em reuniões diárias (dailys), além de melhoria contínua baseada em retrospectivas. A proposta é orientar o usuário de forma prática e personalizada.
Por exemplo, o assistente pode analisar os hábitos de consumo e sugerir ajustes: “Observei que você tem gastado bastante com iFood. Para o próximo mês, que tal reduzir esse valor em 30%? Assim, você poderia direcionar essa economia para a poupança da sua viagem para a Itália.”
Além disso, o assistente oferece projeções financeiras: “Se você economizar 30% desse valor mensalmente, em 7 meses terá acumulado X reais para sua viagem.”
Diferente de métodos tradicionais, como anotações em papel, o assistente registra automaticamente as informações no dia a dia e utiliza conceitos matemáticos para gerar estimativas futuras, ajudando o usuário a tomar decisões mais conscientes e alcançar seus objetivos financeiros.
